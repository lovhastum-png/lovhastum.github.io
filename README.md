# 情绪休息站

一个可直接运行的纯静态交互网站。整站以深夜溪流、暖金微光和低对比界面为统一视觉语言。

## 运行方式

优先方式：直接双击 `index.html`。

如果浏览器对本地文件功能有限，可在项目目录启动任意静态服务器，例如：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后打开 `http://127.0.0.1:4173/`。

项目没有安装依赖、构建或联网要求。

## 主要页面

- 首页：三个主入口与五项主导航
- 探索：51 条心理与行为小知识夜读书
- 情绪调配：八种试剂、实时液体、归一化比例、搅拌与配方保存
- 灵感小站：静息区、标签回收口、补给柜、安静书案、释放室、今日提示卡和值班窗口
- 关于我们：项目定位与使用边界
