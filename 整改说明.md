# 《情绪休息站》整改说明

## 本次完成内容

1. 信息架构
   - 主导航收敛为：首页 / 探索 / 情绪调配 / 灵感小站 / 关于我们。
   - 首页三个入口分别进入感觉选择、情绪调配和模块索引。
   - 所有具体模块增加显式返回入口，并保持主导航高亮一致。

2. 情绪调配室
   - 使用八种情绪试剂，不使用滑杆。
   - 每次点击加入一份，比例按总份数归一化，并用最大余数法保证显示总和为 100%。
   - 玻璃容器为独立前景交互对象；液面随份数上升，并具有渐变、扩散、波纹、气泡和旋涡。
   - 新试剂加入时显示液流和局部综合色扩散。
   - 搅拌棒从上方插入，在液体内部沿小范围圆周运动后退出，不做钟表式旋转。
   - 配方报告包含名称、比例、说明、建议去向、非诊断声明、保存与摘要复制。

3. 静息区
   - 改为 4 秒吸气、2 秒停留、6 秒呼气的状态机。
   - 支持鼠标按住、触摸按住和空格键。
   - 明确显示阶段、倒计时、操作提示和完成轮数。
   - 页面始终只有一个主气泡；吸气缩小，呼气恢复并产生三层涟漪。

4. 标签回收口
   - 删除点击后自动回收逻辑。
   - 使用 Pointer Events 实现鼠标和触摸真实拖拽，保留抓取点偏移。
   - 拖入回收篮时高亮；命中后缩小落入、篮子弹动、光点与计数同步更新。
   - 未命中时使用缓动回弹原位。
   - 完成状态为“已放下 3 / 3”。

5. 安静书案
   - 提笔写字：纸面输入、字数限制、保存、清空和换纸重置均可用。
   - 磨墨静心：必须按住砚台真实画圈，墨色与进度随轨迹增加，完成后才能回到纸面。
   - 点灯片刻：点击灯盏改变灯焰、光晕和整页暖光状态。
   - 模式切换只更新模块状态，不重建页面。

6. 今日提示卡
   - 使用 `transform: rotateY()`、`preserve-3d` 与背面隐藏实现轻量翻牌。
   - 连续抽取时先回到背面、更新内容，再翻到正面。
   - 提示语、主题、标签和剩余次数在固定时机更新，不重建整页。
   - 使用“今日提示卡 / DAILY REST CARD”命名，不含塔罗或占卜表达。

7. 值班窗口
   - 删除大头卡通值班员，改为克制的小剪影和窗边杯子。
   - 增加木质承台、接触阴影和空间层次，弱化贴图悬浮感。
   - 右侧改为低对比“窗边纸条”，默认与实时内容最多显示三条。
   - 快捷入口精简为两个；输入限制长度并可实际提交。

8. 补给柜与释放室
   - 补给柜支持格子点亮、领取补给卡并保存到今晚记录。
   - 沙袋支持点击和水平拖动，带方向、衰减摆动、链条整体跟随、阴影位移和次数记录。

9. 探索页
   - 知识内容由 6 条扩展为 51 条。
   - 每条包含中文名、英文名、简短解释、研究者、提出背景、经典实验或例子、概念发展、生活例子和温和提示。
   - 保留夜间展开书本结构，目录可滚动，上一条 / 下一条直接更新书页内容。

## 性能优化

- 背景图片已移出页面模板，改为 `index.html` 中两张常驻背景缓冲层；普通交互不会删除、重建或重设背景节点。
- 只有路由真正变化时才允许写入下一张背景的 `src`，并在解码完成后交叉淡入。
- 十张核心背景在 HTML 和脚本初始化阶段双重预加载。
- 页面切换等待新背景解码完成后再以 `opacity + transform` 交叉淡入，避免黑白屏。
- 试剂、翻牌、书案、拖拽、补给、沙袋和留言均改为局部 DOM 更新，不再销毁整页。
- 隐藏/离开的页面停止计时器和 Canvas 动画；离场萤火虫暂停。
- Canvas 动画与鼠标视差使用 `requestAnimationFrame`。
- 去除全屏背景滤镜，减少大面积重绘；主要玻璃面板改用半透明背景、描边和阴影。
- 拖拽、翻牌、沙袋和页面切换主要使用 `transform` 与 `opacity`。
- 移除通过读取 `offsetWidth` 强制重启动画的同步布局操作。

## 补给柜与值班窗口追加修复

- 夜间补给柜的选择区、柜体、格口与记录区改为高不透明度实体层，并增加木质内框、暖光与接触阴影。
- 值班窗口不再加载大头值班员素材，也不再使用在线客服式标题和消息气泡。
- 留言区改为有木框、顶梁和落地阴影的“窗边留言笺”，输入动作命名为“递出纸条”。
